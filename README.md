# How to Use

- [kakao developers console](https://developers.kakao.com/console/app)에서 애플리케이션 추가 후 앱 키 탭에서 REST API 키 정보 얻기
- `.env` 파일을 생성해 `KAKAO_REST_API_KEY`로 얻은 REST API 키 정보 추가
- `pip install -r requirements.txt`
- `python main.py`
